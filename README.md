# Simple Discord Bot

## Dependences

- discord.js
- dotenv

### Install Dependences

```shell
npm install
```

## Running
 There is two ways to run the project on script ``package.json``

On node:
```shell
npm run start
```

On dev:
```shell
npm run dev
```

### Config Enveroment
Configure ``.env`` with enveroments variables described on ``.env.example``
